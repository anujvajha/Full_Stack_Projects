# Mini E-Commerce App 

A simple full-stack e-commerce practice project built using MERN stack.  
This application allows users to register, log in, browse products, add items to cart, and simulate checkout.

## 🚀 Tech Stack

- **Frontend:** React, Axios, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** JWT (JSON Web Tokens), Cookies
- **Other:** bcrypt for password hashing

## ✨ Features

- User authentication (Signup / Login / Logout)
- Protected routes using JWT
- Product listing
- Add to cart functionality
- View and delete cart items
- Basic checkout simulation with total price & tax calculation
